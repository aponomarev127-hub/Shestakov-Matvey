# Currency Converter

Автор: Шестаков Матвей Васильевич 

## Описание

Currency Converter — это приложение с графическим интерфейсом на Python (Tkinter), предназначенное для конвертации валют по актуальному обменному курсу. Программа позволяет быстро переводить сумму из одной валюты в другую с использованием внешнего API, отображающего реальные курсы валют.

Пользователь может выбрать исходную и целевую валюту, ввести сумму для обмена и мгновенно получить результат. Также приложение сохраняет историю всех операций в JSON-файл и автоматически загружает её при следующем запуске.

## Возможности

- выбор валюты "из" и "в"
- ввод суммы
- кнопка конвертации
- получение актуального курса через API
- таблица истории операций
- сохранение данных в JSON
- загрузка истории при запуске
